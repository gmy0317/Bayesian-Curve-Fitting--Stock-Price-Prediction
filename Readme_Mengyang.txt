Bayesian Curve Fitting based Stock Price Prediction
Mengyang Guo, Civil and Environmental Engineering
#######################################################
This is a Eclipse project and you can run the code in Eclipse by importing projects from file system.


The Jar file is under the OUT folder. The Google Stock Price data is under the OUT\GOOG folder.
Please use the following command to run the code in  command line:

java -jar Curve_Fitting.jar GOOG\Google_d1.csv
java -jar Curve_Fitting.jar GOOG\Google_m1.csv
